# DThorsonP GitHub Profile Repository

## Overview
This is a GitHub profile README repository containing personal profile information and Windows PowerShell scripts for Apache Tomcat upgrade automation.

## Project Structure
- `README.md` - GitHub profile README with personal introduction
- `upgradeTomcat-*.ps1` - Windows PowerShell scripts for automating Apache Tomcat upgrades

## Contents

### PowerShell Scripts
These scripts are designed to run on Windows systems to automate Apache Tomcat 9.x upgrades:
- Stop existing Tomcat service
- Uninstall old service
- Download and extract new Tomcat version
- Copy webapps and logs from old installation
- Update environment variables
- Install and start new Tomcat service

## Notes
- This is a documentation/script repository, not a runnable application
- The PowerShell scripts are Windows-specific and require administrator privileges
- Scripts are meant to be executed in a Windows environment, not in Replit
